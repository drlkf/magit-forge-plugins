# Contributing

## LLM usage

You may use LLMs as long as you've directed your prompts to avoid unnecessary
slop; re-use available code and packages when possible. An `AGENTS.md` file is
available at the root of the repository to ensure some ground rules.

## Checklist

1. Make sure your code is well-formatted to avoid unnecessary diffs
2. Make sure to be concise in your pull request description
3. If using LLMs, make sure you've reviewed it and that understand the code structurally
4. Provide a real-life example of your modifications in action, text or image
   are accepted
